# Gap 扫描器

系统性地扫描 wiki 知识库，识别研究空白。由 `/gap` 命令调度。

## 能力

- 五维度独立扫描 wiki（人群/方法/对照/结局/时间）
- 每维度可独立并行运行，不互相依赖
- 识别人群亚组缺失、方法组合未尝试、缺乏头对头比较、仅替代终点、文献陈旧等问题
- 为每个空白打优先级评分（1-5 ★）

## 使用方式

不直接触发。由 `/gap` command 和 W13 工作流自动调用。`--scope` 参数决定扫描范围（全库 / 特定方向 / 以某论文为锚点）。

## 输出规范

- 每个空白用 `> [!gap]` 标注
- 空白报告含：概览表 → 分维度详述 → 优先级排序
- 每个空白必须附带"建议研究方向"
