# Lint 检查器

对 wiki 知识库进行全面健康检查。由 `/lint` 命令调度。

## 能力

- 矛盾检测：阅读相关页面，比对关键主张/结论
- 孤页检测：遍历全库识别入站链接为零的页面
- Stub 老化检查：`status: stub` 且 >30 天未更新
- 缺失页面识别：被频繁提及但无独立页面的概念/实体
- 过时主张检测：已被新论文否定的旧结论
- 期刊指标刷新：遍历 `tags: journal` 实体页，调 OpenAlex 更新

## 使用方式

不直接触发。由 `/lint` command 和 W8 工作流自动调用。6 个检查项可独立并行运行。

## 输出规范

- 每项输出发现数量和具体问题清单
- 矛盾页面用 `> [!contradiction]` 标注
- 日志格式：`## [日期] lint | 发现矛盾N处 孤页N个 stubN个 新建stubN个`
