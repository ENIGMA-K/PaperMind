# /gap — 研究空白分析

系统识别当前 wiki 知识中的研究空白。

## 参数

- `$ARGUMENTS`：`[--scope <方向|论文>]`
- 示例：
  - `/gap` → 全库空白扫描
  - `/gap --scope "CABG术后桥血管评估"` → 聚焦该方向
  - `/gap --scope [[Zhang 2023 - PD-1 in NSCLC]]` → 以该论文为锚点找空白

## 执行流程

严格按 CLAUDE.md W13 执行：确定范围 → 五维度扫描（人群/方法/对照/结局/时间）→ 生成空白报告 → 优先级排序。

### 五维度扫描

| 维度 | 检查 |
|------|------|
| 人群空白 | 哪些人群/亚组缺乏研究？（老年、女性、中国人群、特定合并症） |
| 方法空白 | 哪些方法组合未被尝试？缺乏金标准验证？ |
| 对照空白 | 缺乏头对头比较？缺乏 vs 当前最佳实践的对照？ |
| 结局空白 | 只有短期终点？缺乏长期随访？缺乏患者报告结局？ |
| 时间空白 | 哪些领域的文献陈旧（>5 年无新证据）？ |

### 输出

`wiki/synthesis/Gap分析_<范围>_YYYY-MM-DD.md` — 每个空白用 `> [!gap]` 标注，带优先级评分（1-5 星）。
